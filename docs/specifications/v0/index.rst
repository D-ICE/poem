POEM File Format Specification (v0)
===================================

.. note::
   The **POEM File Format Specifiations** v0 only define RULES to manage PPP performance
   polar tables, i.e. for Power Prediction with Speed control.
   Specifications are given for 5-Dimensional PPP performance polar tables currently used at D-ICE ENGINEERING.
   The future version will provide much more flexibility.
